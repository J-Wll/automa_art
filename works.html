How this website works

This website was created using html, css and javascript; html is used to place the basic information, like this text. The css code then takes this text and applies my pretty styling to it, turning the text white and the background grey.

The javascript is used to power the generators on the home page, both generators run on the same foundation, heavily modified code I found here(the creating a loop section).

The code generates triangles of a slightly randomised size in either an orange or blue colour before turning a randomised amount and placing another triangle. In order to run cleanly it has to refresh the page to fully clear the canvas.
