The process of development

Stage one of developement was brainstorming ideas, creating a rough plan and placing it in the sturcture on trello, then I created a rough version of the layout using a website called draw.io.

Stage two involved learning the languages needed and then creating the basic html and css structure while trying to understand and create the javascript outside of course hours.

Stage 3 was putting it all together and fixing all my mistakes.
